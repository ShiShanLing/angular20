炫酷图表演示

独立模块 · ECharts 5 + ECharts-GL WebGL。含省域热力、直角 3D 曲面/立体柱、2D 飞线与3D 地球；另有象形柱、水球图、主题河流等示意。

省域热力 · 迁徙弧线

map + visualMap 分级着色；叠加少量暖色 lines（慢速 pin 尾迹、较高曲率），与下方青蓝飞线区分。示意数值由行政区名称哈希生成。

直角 3D · 曲面 + 立体柱 + 散点

grid3D：surface 函数波面 + bar3D 栅格楼块 + scatter3D；可拖拽旋转、自动漫游。与 globe 球坐标场景不同。

2D 中国底图 · 流光飞线

geo + lines（effect 拖尾箭头）+ effectScatter 城市涟漪；数据为示意航线。国界 GeoJSON 来自阿里云 DataV（需联网）；亦可改为本地静态资源。

象形柱图 · Pictorial Bar

pictorialBar：symbolRepeat 堆叠 + 自定义 path:// 象形符号（与底层满格对比）。

水球图 · Liquid Fill

扩展 echarts-liquidfill + type: 'liquidFill'；双球多层波纹（示意百分比）。

主题河流 · Theme River

singleAxis + themeRiver，多品类流量在时间上堆叠成「河带」，适合展示成分随时间的演变（示意数据）。

柱状排序竞速 · Bar Race

timeline 自动播放 + bar.realtimeSort；每个时间节点重排横向柱状（示意框架热度，非真实排名）。

日历热力图 · Calendar Heatmap

calendar + heatmap，展示全年单日强度（示意 2026；可拖拽 visualMap 区间）。

3D 地球 · WebGL（可拖拽旋转）

GlobeComponent + scatter3D / lines3D；星空全景 + 城市灯光 + 飞线；天球壳上另有日月与河外星系示意（非真实尺度与方位）。行星贴图 Three.js；星空 starfield 走 GitHub raw。

霓虹流光 · 多层渐变面积折线

lineStyle shadowBlur + 纵向线性 areaStyle，模拟实时监控感

霓虹平行坐标

parallel + visualMap 映射 line 颜色；多维折线束，悬停加粗单条（示意 profiling）。

三联 HUD 仪表盘

三个 gauge 并排：progress + roundCap，无指针圆角弧 + 数值动画。

涟漪散点

effectScatter + visualMap 映射粒子大小与色相

南丁格尔玫瑰

roseType: 'area' + 径向渐变填色

旭日图 · 层级辐射

sunburst · 点击扇区可强调祖先链

力导向关系图

graph + force，节点可拖拽、可缩放平移画布

桑基图 · 流量分解

lineStyle.color: 'gradient' 沿线着色

矩形树图

treemap 层级钻取 + 圆角分块
